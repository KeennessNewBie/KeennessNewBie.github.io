# [206. 反转链表](https://leetcode.cn/problems/reverse-linked-list/)

## 两种解法

### 循环

1. 三个节点 pre cur tmp
2. 保存当前节点的下一个节点到 tmp
3. 将当前节点的 next 指向上一个节点 cur.next = pre
4. 将 pre 和 cur 都向后移动
   code：

   ```java
   class Solution {
    public ListNode reverseList(ListNode head) {
        ListNode tmp = null;
        ListNode cur = head;
        ListNode pre = null;
        while(cur !=null){
            tmp = cur.next;
            cur.next = pre;
            pre = cur;
            cur = tmp;
        }
        return pre;
    }
   ```
